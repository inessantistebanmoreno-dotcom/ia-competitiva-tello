Añadir nuevo competidor
Requiere desarrollo de un scraper a medida

Añadir un competidor implica crear un scraper específico para su web. Cada web tiene una estructura HTML diferente que hay que analizar e implementar manualmente.

Para añadir un competidor, necesitas:
① Crear scrapers/nuevo_scraper.py con la clase del scraper
② Registrarlo en scrapers/__init__.py y config.py
③ Añadir los colores y la etiqueta en el dashboard
④ Configurar el scraping en GitHub Actions

¿Qué competidor quieres añadir? Compártelo con el equipo de desarrollo con la URL de su catálogo de productos.
